# posting--page

Uma página web estática que simula a criação de posts construída com HTML, CSS, e JS. 

A página comunica com o API aberto [JSONPlaceholder](https://jsonplaceholder.typicode.com/) para simular a comunicação com um servidor.

## Instalação

1. Em seu terminal de escolha, clone o repositório com `git clone https://github.com/arthur-tsutiya/posting--page`.
2. Mude o seu diretório para a pasta do projeto com `cd posting--page`.
3. Abra o arquivo `index.html` em seu navegador de escolha.
